# Персональное портфолио (визитная карточка) разработчика

Интерактивное веб-приложение для демонстрации профессиональных навыков, опыта работы и проектов. Разработано с использованием современного технологического стека.

## 🚀 Технологический стек
- **Фреймворк:** React 19 с TypeScript
- **Сборщик:** Vite
- **Стилизация:** Tailwind CSS v4
- **Иконки:** Lucide React
- **Оптимизация:** Static Generation (SSG), Canvas particles, интерактивный Terminal

## ✨ Особенности
- **Интерактивный терминал:** Эмулятор командной строки Bash с поддержкой команд `help`, `skills`, `clear`, `about` и пасхалками.
- **Шоукейс проектов:** Динамическая фильтрация и модальное окно с детальным описанием архитектуры.
- **Интерактивная матрица навыков:** Вкладки с опытом работы, ключевыми навыками и образованием.
- **Защита от спама:** Математическая капча на форме обратной связи.
- **Прямые ссылки на медиа:** Интеграция с Telegram, VKontakte, Kinopoisk, Yandex Music, MAX.
- **Адаптивность и доступность:** Идеальное отображение на мобильных, планшетных и десктопных экранах. Полное соответствие стандартам WCAG.

## 📦 Локальный запуск
1. Установите зависимости: `npm install`
2. Запустите локальный сервер для разработки: `npm run dev`
3. Сборка статического сайта: `npm run build`

## 🔗 Деплой на GitHub Pages
Сайт спроектирован для автоматического деплоя на GitHub Pages.
Достаточно сделать пуш в ветку `main` или настроить GitHub Action для автоматической публикации.
